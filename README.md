
## アプリケーシション
初心者向けの家庭菜園アプリ

## アプリケーション概要
ユーザーターゲット :家庭菜園を始めたばかりの初心者

アプリケーション概要 :「苗を植えたけど育て方が合っているか不安」「枯れてしまったけど原因がわからない」といった悩みを持つ初心者が、他のユーザーのリアルな育成記録や体験談を通じて学べるサポート型SNSアプリです。野菜の育成経過を記録したり、失敗や工夫を投稿することで、ユーザー同士が気軽に知識や経験を共有し合える安心感のあるコミュニティを提供します。

## 利用方法
1. URLにアクセスし、ヘッダー右上の「ログイン」ボタンからログイン画面に移動します。
2. テスト用アカウントを使用しログインします。
3. 新規投稿ボタンから項目を入力し「投稿する」ボタンから新規呟きを投稿します。
4. 投稿内容には画像1枚と文章を含めることができます。
5. 他のユーザーの呟きを閲覧し、コメントを残すことも可能です。
6. 投稿した呟きはトップページから編集・削除できます。
7. トップページにある作り方ガイドを押すと、野菜の名前が表示され、それぞれ野菜ごとの育て方一覧が確認できます。

## アプリケーションを作成した背景
解決したい課題 :同じ目線のリアルな体験を共有できる場を作る

私自身が家庭菜園を趣味として始めた際に、インターネットや書籍で得られる情報が豊富である一方、「自分のやり方で本当に合っているのか」「失敗したけれど、これはよくあることなのか」といった不安を抱えることがありました。プロや経験者の情報は参考になる一方で、かえってハードルの高さを感じることもありました。
その経験から、初心者が初心者同士で気軽に育成記録や失敗・工夫を共有し合える場の必要性を強く感じました。ただ育てるだけでなく、「育てている過程を安心して見せ合える」「仲間がいると感じられる」仕組みを通して、より楽しく・継続しやすい家庭菜園の環境を提供したいと考えました。

## ローカルでの動作方法

1. リポジトリをクローンします。
```ruby
  git clone https://github.com/okada076/original_app-42380
```

2.必要なGemをインストールします。
  ```ruby
  bundle install 
  ```
3.データベースを設定します。
  ```ruby
  rails db:create
  rails db:migrate
  ```
4.アプリケーションを起動します。
  ```ruby
  rails s
  ```
5.ブラウザで http://localhost:3000 にアクセスします。


## 工夫したポイント
家庭菜園初心者でも気軽に育成記録を残せることを目指して設計されています。特に以下の点に工夫を凝らしました。
<br> - 投稿時の簡単な画像アップロード：育てている野菜の写真を手軽にアップでき、成長の様子を記録できます。
<br> - 選べるカテゴリー（育成記録／つまずきノート）：成功体験だけでなく失敗も共有できることで、初心者が安心して学べる環境を提供します。
<br> - タグ機能（予定）：育成環境や工夫した点などをタグで分類し、情報共有をより柔軟に行えるようにします。
<br> - コメント機能（予定）：他のユーザーの投稿に対して感想や質問を気軽に書き込めるようにし、ユーザー同士の交流を促進します。

## 今後の課題
現在のプリは基本的な機能を提供していますが、ユーザーのニーズに応えるために、以下の追加機能を検討しています。
<br> -「いいね」一覧表示機能：ユーザーが過去に「いいね」した投稿を一覧で確認できるページを設け、お気に入りの記録を振り返りやすくします。
<br> - 画像の複数アップロード：複数の画像を一度に投稿できる機能を追加します。
<br> - 投稿データの可視化・分析：投稿された野菜の種類や数を集計し、「人気の野菜ランキング」や「季節ごとの投稿傾向」などをグラフで可視化する機能を検討しています。
